Hadoop TF-IDF MapReduce Job
Overview
This project involves writing a MapReduce job for Hadoop to compute the Term Frequency-Inverse Document Frequency (TF-IDF) metric for words across a set of documents. The implementation is done using Python 2, employing Hadoop's Streaming API for executing map and reduce tasks. This method helps to determine the relevance of words in documents, which is crucial for searches based on term queries.

The TF-IDF metric combines:

TF (Term Frequency): The frequency of a word in a document.
IDF (Inverse Document Frequency): A measure of how much information the word provides, i.e., whether it is common or rare across all documents.
For further understanding of TF-IDF, refer to the Wikipedia article on TF-IDF (as of January 23, 2024).
